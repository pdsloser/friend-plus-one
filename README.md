# 朋友加一：6人聚

固定六人的小型主題交流活動平台 MVP。

## Tech Stack

- Next.js App Router
- TypeScript strict
- Tailwind CSS
- Supabase Auth/PostgreSQL/RLS
- Vitest
- Playwright

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run lint
npm run build
npm run test:e2e
```
